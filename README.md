## Introduction

This tool assists in creating Fortnite Galaxy Swapper v2 Plugins for skin swapping using Python. The `main.py` file serves as the core script for generating customized plugins.

## Getting Started

To use this tool, ensure you have the following prerequisites:

### Prerequisites

- Python 3.x installed on your system.
- Required modules:
    - `requests`
    - `json`

Install the necessary modules using the following command:

```bash
pip install requests json
```

## File Structure

- `main.py`: The main Python script for generating plugins.
- `Templates/`: Directory containing template files.
- `Exports/`: Folder where exported JSON files are stored.
- `Plugins/`: Directory where generated plugins will be saved.

## Usage

1. Run `main.py`.
2. Select the desired skin swap option.
3. Enter the IDs for the skins you want to swap.
4. Optionally, add a message for the swap.
5. The generated plugin will be stored in the `Plugins` directory.
